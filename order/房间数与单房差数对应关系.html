单房差数只与成人游客数有关，与儿童、婴儿数无关

定义：成人数M，房间数N，单房差数A

成人数M——最小值为1；

计算：

场景举例：

一、小夫，小妻是一对情侣，在旅游途中结识了2位互不相识的朋友；之后4人一同游玩。

傍晚4人一同决定让小夫在收客通中预定另一条游览线路。

则一同4人、需要3间房，则需要补的单房差数为：


A = 房间数3 * 2 - 成人数4 = 2


二、7人，需要6间房，则需要补的单房差数为：


A = 房间数6 * 2 - 成人数7 = 5



最小房间数 Nmin：

成人数M为奇数时  Nmin = （成人数M+1）/2

成人数M为偶数时  Nmin = 成人数M /2

最大房间数 Nmax：

                           Nmax =   成人数M

房间数       Nx：

                           Nmin ≤ 房间数Nx ≤ Nmax

单房差数     A :

                         A =  房间数Nx * 2- 成人数M
